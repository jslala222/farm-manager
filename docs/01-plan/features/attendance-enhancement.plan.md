# [Plan] 출근부 고도화: 과거 기록 소급 및 알바 변동 급여 시스템

## 1. 개요 (Overview)
농장 운영 현장에서 발생하는 "며칠 전 출근 기록 누락" 및 "알바생별 매일 다른 근무시간과 일당" 문제를 해결하기 위해 시스템을 고도화합니다.

## 2. 목표 (Goals)
- **소급 적용**: 오늘뿐만 아니라 달력을 통해 과거 날짜를 선택하여 출근부 기록/수정 가능.
- **유연한 급여/시간**: 알바생 직군에 대해 개별 일당 및 근무시간 수동 입력 허용.
- **팀 단위(인원수) 기록**: "김순애 외 4명" 같은 팀 단위 출근을 위해 `인원수(headcount)` 필드 도입 (통계 정확도 확보).
- **보편적 적용**: 전 직군(사장/가족/직원/알바)에 대해 특이사항 메모 기능 제공 및 신규 가입자 소급 적용.

## 3. 주요 기능 (Features)
- [ ] **Date Picker (달력)**: 출근부 상단 날짜 선택 기능.
- [ ] **Dynamic Wage Input**: 알바생 전용 일당/시간 입력 필드.
- [ ] **Attendance Notes**: 전 직군 공통 특이사항 메모 필드.
- [ ] **Attendance Statistics**: 일/주/월별 인원 및 노무비 통계 대시보드.
