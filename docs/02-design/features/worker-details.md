# PDCA 설계서 (Design - D)

## 1. 개요
근로자 인적 사항 확장 요청에 따라 DB 구조와 관리 UI를 새롭게 설계합니다.

## 2. 주요 설계 및 변경 사항

### 2.1 데이터베이스 (Supabase)
`workers` 테이블에 다음의 컬럼을 추가/복구합니다.
- `phone`: 연락처 (기존 누락분 복구)
- `gender`: 성별 (`male`, `female` 구분)
- `address`: 거주 주소
- `notes`: 특이사항 및 메모 (장문 가능)

### 2.2 UI 컴포넌트 설계 (WorkersPage)
- **성별 선택**: 버튼 토글 형태(남성/여성)로 직관적인 선택 제공.
- **주소 입력**: 한 줄 텍스트 입력 칸 추가.
- **메모 란**: 텍스트 에어리어(Textarea)를 활용하여 충분한 기록 공간 확보.
- **카드 뷰 확장**: 목록에서도 성별과 메모 요약이 보이도록 카드 디자인 수정.

## 3. 구현 파일 목록
