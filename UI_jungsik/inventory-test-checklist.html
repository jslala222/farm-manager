2안+ 테스트 체크리스트

재고관리 검증 문서

이 문서는 현재재고 + 직접조정 + 조정이력 + 판매 자동 차감 구조를 실제 운영에서 검증하기 위한 체크리스트입니다. 택배, 납품, 모바일, PC 기준으로 나누어 점검할 수 있게 구성했습니다.

1. 공통 기본 검증

항목 검증 내용 결과
재고 ON/OFF설정에서 ON/OFF 변경 시 화면/검사 동작이 의도대로 바뀌는가
현재재고 표시실데이터 기준 현재재고가 누락 없이 표시되는가
단위 표시kg / 박스 / 병 / 개 등 단위가 올바르게 보이는가
부족 경고음수 또는 부족 상태가 명확히 보이는가
조정 후 반영직접조정 저장 후 현재재고가 즉시 갱신되는가

2. 직접조정 검증

조정 유형 검증 내용 결과
폐기폐기 수량 입력 시 현재재고가 정확히 감소하는가
부분폐기일부 수량만 폐기했을 때 남은 수량이 정확한가
전부폐기전체 수량 폐기 시 0으로 반영되는가
가공전환원물 가공전환 시 원물 재고가 정확히 차감되는가
수동보정(+)플러스 보정이 재고에 반영되는가
수동보정(-)마이너스 보정이 재고에 반영되는가
사유 저장조정 사유가 이력에 정확히 남는가

3. 조정이력 검증

항목 검증 내용 결과
일시조정 시각이 정확히 기록되는가
품목명어떤 품목이 변경됐는지 명확한가
이벤트 유형폐기 / 가공전환 / 보정 등이 정확히 표시되는가
수량변경 수량이 맞는가
사유입력한 사유가 그대로 보이는가
작성자입력자 정보가 필요 시 추적 가능한가

4. 택배 판매 검증

상황 검증 내용 결과
정상 판매재고가 충분할 때 저장이 정상 완료되고 재고가 차감되는가
부족 판매재고 부족 시 shortage dialog가 의도대로 뜨는가
경고 모드경고 후 계속 저장 시 저장은 되고 재고는 차감되는가
차단 모드차단 설정 시 저장이 막히는가
가공품 판매가공품 판매 시 가공품 재고만 차감되는가

5. 납품 판매 검증

상황 검증 내용 결과
보류 저장납품 보류 저장 시 재고 차감 정책이 의도대로 동작하는가
즉시 정산 저장즉시 정산 시 재고가 차감되는가
부족 경고부족 시 모달 내용이 품목/수량 단위와 함께 정확한가
계속 저장경고 모드에서 계속 저장 흐름이 끊기지 않는가
수정 저장기존 납품 내역 수정 시 재고 재계산이 맞는가

6. 모바일 검증

항목 검증 내용 결과
숫자 키패드수량/가격 입력 시 숫자 키패드가 잘 뜨는가
버튼 가림모달/바텀시트 하단 버튼이 가려지지 않는가
스크롤재고 목록과 모달 내부 스크롤이 자연스러운가
탭 전환원물/가공품/부족 필터 전환이 편한가
직접조정 입력폐기/가공전환/보정 입력이 손쉽게 가능한가

7. PC 검증

항목 검증 내용 결과
한눈에 보기재고 요약과 목록이 한 화면에서 잘 읽히는가
테이블 가독성품목, 수량, 단위, 상태가 혼동되지 않는가
조정이력 조회필터/정렬 없이도 기본 조회가 충분히 이해 가능한가
멀티 작업재고 확인 후 판매 화면으로 이동하는 흐름이 빠른가

8. 실데이터 검증 포인트

검증 포인트 설명 결과
경준싱싱농장 기준 재고실제 입력된 원물/가공품 데이터가 문서상 가정과 크게 다르지 않은가
원물과 무관한 가공품독립 가공품도 무리 없이 관리 가능한가
실제 직원 사용성관리자가 아니라 사용자/직원이 입력하기에 부담이 적은가
조정 빈도폐기/보정/가공전환 입력 빈도가 과도하지 않은가
최종 체크 기준: 재고관리는 정확성도 중요하지만, 현장에서 계속 쓸 수 있는지가 더 중요합니다. 테스트는 계산 정확성뿐 아니라 입력 부담, 모바일 사용성, 판매 흐름 방해 여부까지 함께 확인해야 합니다.
