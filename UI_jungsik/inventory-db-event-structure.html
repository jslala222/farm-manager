2안+ DB / 이벤트 구조안

재고관리 데이터 구조 문서

이 문서는 현재재고 + 조정이력 분리형을 실제 구현 가능한 수준으로 내린 초안입니다. 목표는 처음부터 거대한 ERP를 만드는 것이 아니라, 현재 재고를 안정적으로 보여주고 직접 조정할 수 있게 하는 구조를 잡는 것입니다.

핵심 구조: 원물 재고와 가공품 재고는 기본 분리
핵심 로직: 현재재고는 조회용 집계 관점, 조정이력은 원인 기록 관점
확장 포인트: 필요 시에만 원물 → 가공전환을 연결

1. 전체 구조 요약

구성 요소 역할 비고
원물 재고 데이터 현재 원물 수량 / 등급별 수량 관리 핵심
가공품 재고 데이터 현재 가공품 수량 관리 핵심
조정이력 데이터 왜 재고가 바뀌었는지 저장 핵심
판매 데이터 택배 / 납품 저장 시 재고 차감 근거 기존 연동
가공 연결 데이터 원물 → 가공전환 연결 정보 선택 기능

2. 최소 테이블 관점

테이블 / 개념 목적 초기 도입 여부
`harvest_records` 수확 입력 원본 기존 사용
`sales_records` 택배 / 납품 판매 원본 기존 사용
`inventory_adjustments` 폐기 / 가공전환 / 보정 / 등급변경 이력 저장 즉시 확장
`inventory_current_view` 또는 집계 로직 현재 재고 계산 결과 즉시 필요
`processed_goods` 개념 또는 상품구분 필드 가공품 품목 구분 선반영 가능

3. 현재재고 계산 기준

재고 종류 증가 요인 감소 요인 비고
원물 재고 수확 입력, 수동보정(+) 판매, 폐기, 가공전환, 수동보정(-) 등급별 재고 가능
가공품 재고 직접 입고, 생산등록, 수동보정(+) 판매, 폐기, 수동보정(-) 원물과 독립 가능

4. 조정이력 이벤트 표준안

event_type 의미 필수 필드 예시
`manual_adjust` 재고 수량 직접 보정 품목, 수량, 증감, 사유 실물 수량 확인 후 -2
`waste_out` 폐기 차감 품목, 수량, 사유, 등급(선택) 중 3박스 폐기
`process_out` 가공용 전환 차감 품목, 수량, 사유 딸기 10kg 가공용 전환
`grade_change` 등급 이동 품목, from_grade, to_grade, 수량, 사유 특/상 5박스 → 중 5박스
`stock_in` 직접 입고 또는 가공품 입고 품목, 수량, 사유 딸기잼 20병 생산등록

5. 조정이력 필드 초안

필드 설명 필수 여부
`farm_id` 농장 구분 필수
`crop_name` / `item_name` 품목명 필수
`item_type` 원물 / 가공품 필수
`event_type` 폐기 / 가공전환 / 보정 / 등급변경 필수
`quantity` 변경 수량 필수
`unit` kg / 박스 / 병 / 개 필수
`from_grade` 변경 전 등급 선택
`to_grade` 변경 후 등급 선택
`reason` 사유 권장
`created_by` 입력자 권장
`created_at` 입력 시각 필수

6. 원물 / 가공품 분리 방식

원물 재고 설명
입고 기준 수확
감소 기준 판매 / 폐기 / 가공전환 / 보정
등급 관리 가능
현재재고 화면 등급별 수량 표시 가능
가공품 재고 설명
입고 기준 직접입고 / 생산등록 / 보정
감소 기준 판매 / 폐기 / 보정
원물 연결 필수 아님, 선택 사항
현재재고 화면 독립 품목으로 표시

7. 선택 연결 구조

상황 처리 방식 데이터 반영
원물과 무관한 가공품 독립 가공품으로만 관리 가공품 재고만 증감
원물을 가공용으로 사용 `process_out` 기록 원물 재고 차감
가공품 생산 결과도 기록하고 싶음 `stock_in` 기록 가공품 재고 증가
레시피 환산까지 필요 후속 확장 초기 범위에서는 제외

8. 즉시 구현 / 후속 확장

구분 내용
즉시 구현 현재재고 계산, 조정이력 저장, 폐기, 가공전환, 수동보정, 판매 자동 차감
선반영 가능 등급변경, 가공품 독립 재고, 입력자 추적
후속 확장 원물-가공품 자동 연결, 레시피 기반 환산, 고급 리포트
결론: 2안+의 DB 구조는 복잡한 자동 추적보다, 현재재고를 안정적으로 보여주는 계산 구조와 조정이력 이벤트 저장 구조를 먼저 확보하는 데 집중해야 합니다. 이렇게 하면 원물/가공품 기본 분리와 선택 연결을 모두 수용할 수 있습니다.
