# 단축 명령어 (Chat Shortcuts)

## /비키트
> 비키트와 에이전트 긴급회의 해서 철저히 검수후 3안으로 제안하고 그중에 추천 안 한개 해줘

**용도**: 3개 안 제시 + 추천 1개 선정 (의사결정 가속화)

**사용 예**:
- Chat에서 `/비키트` 입력 후 Tab
- 또는 복사해서 사용

---

## 설정 방법

### Option 1: 직접 입력 (가장 간단)
Chat에서 `/비키트` 그냥 입력하면 됨

### Option 2: VS Code Snippet 활용 (추천)
1. `Ctrl+Shift+P` → "Configure User Snippets"
2. "chatcommands" 검색 후 선택
3. 아래 추가:
```json
{
  "비키트": {
    "scope": "chat",
    "prefix": "/비키트",
    "body": "비키트와 에이전트 긴급회의 해서 철저히 검수후 3안으로 제안하고 그중에 추천 안 한개 해줘"
  }
}
```
